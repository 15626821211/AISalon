# 智能取数模块

## 概述
新增「智能取数」模块，允许用户用自然语言描述业务需求，AI 根据 Hive 表 DDL 元数据自动生成 SELECT SQL，经人工审核后执行查询并提供 CSV 下载。

## 动机
业务人员经常需要从 Hive 数仓提取数据，但不熟悉 SQL 和表结构。通过 AI 自动生成 SQL 可以大幅降低取数门槛。

## 约束
- 仅允许 SELECT 查询，禁止任何 DDL/DML
- 分区表必须走分区过滤，禁止全表扫描
- 查询超时 30 分钟
- 结果限制：50MB / 10万行
- 生成 SQL 必须带库名前缀
- 必须人工审核后才能执行
- 所有登录用户可用

## 外部依赖
- Hive DDL 元数据: MySQL 10.10.106.244:3306/db_bd_prd (t_hive_table + t_hive_database)
- Hive 执行: 10.10.100.146:10000, 账号 hive
- AI: 复用现有 Azure OpenAI (gpt-4o)
